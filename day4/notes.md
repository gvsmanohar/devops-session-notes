
# Vi Editor

## Editing a file

```bash
vi 'filename'

enter insert mode( small i )
type some thing
escape(esc)
save it (:w)
quit(:q!)

save and quit(:wq!)

```

## See line numbers

```bash
:set nu  (see line numbers)
```
## Goto Line number

```bash
:linenumber ( takes cursor to that line)
```

## Search a pattern
```bash
:/pattern (search a pattern and takes to first occurence and type n to got next occurence)
```
